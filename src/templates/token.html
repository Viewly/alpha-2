What is the Viewly token (VIEW)?

VIEW token plays several roles on the Viewly platform:

  • Utility token: Content creators can use the token to publish videos to the Viewly network.
  • Governance token: Token holders can participate in voting that governs the distribution of the tokens, token emissions, worker proposal, etc.
  • Currency: Viewers can support their favorite creators trough social tipping, or purchase items.
VIEW token is an ERC-20 compliant token on the Ethereum blockchain.
